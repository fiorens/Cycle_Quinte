🎼 Cycle des Quintes Interactif
Un outil de théorie musicale interactif conçu pour aider les musiciens à visualiser les relations entre les tonalités, les armures et les degrés harmoniques.

🚀 Démo en ligne : [Accéder au Cycle des Quintes](https://fiorens.github.io/Cycle_Quinte/)

🌟 Fonctionnalités
Roue de navigation interactive : Exploration visuelle des 12 tonalités majeures et mineures.

Analyse Harmonique Automatique : Affiche instantanément les 7 degrés d'une gamme (I, ii, iii, IV, V, vi, vii°) lors de la sélection d'une tonique.

Dual-Notation : Switch instantané entre la notation Anglaise (C, D, E...) et la notation Française (Do, Ré, Mi...).

Visualisation des Armures : Indique précisément le nombre de dièses ou de bémols pour chaque tonalité choisie.

Gestion des Enharmonies : Support complet des tonalités complexes (ex: Fa♯/Sol♭ ou Ré♯m/Mi♭m).

Design Responsive : Optimisé pour une utilisation sur ordinateur, tablette et smartphone.

🛠️ Détails Techniques
Le projet est développé en Vanilla JavaScript, HTML5 et CSS3 sans aucune dépendance externe, garantissant une légèreté et une rapidité d'exécution optimales.

Trigonométrie CSS : Utilisation de cos() et sin() pour le placement dynamique des notes sur le cercle.

DOM Manipulation : Mise à jour dynamique de l'interface et gestion des badges de degrés romains.

Système de Traduction : Moteur de rendu flexible permettant de basculer la notation sans recharger la page.

📖 Comment l'utiliser ?
Sélectionnez une note sur le cercle extérieur (Majeur) ou intérieur (mineur).

Observez les relations : L'outil met en évidence la tonique, ses dominantes, ses sous-dominantes et ses relatifs.

Consultez le tableau : Les accords correspondants à la tonalité s'affichent en bas de page pour faciliter la composition ou l'analyse.

Changez la notation : Utilisez le bouton en haut de page pour adapter l'affichage à votre préférence.
